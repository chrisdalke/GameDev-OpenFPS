The community FPS project...

The OpenFPS community project is one being undertaken by various members of the TGC forums and is open to anyone who wishes to participate. Naturally, the aim is to produce a complete FPS project as a team in the DarkBASIC Pro programming language, as well as with several other tools. This website is for the organization and easy access of source and media for the OpenFPS project.

Biostorm - Chemical Warfare...

The game will be called Biostorm - Chemical Warfare and will feature a campaign and deathmatch styled multiplayer feature. The game is set in modern times, on a the island of a rogue nation. This nation has threatened to bomb other countries with a highly dangerous 'antiprotein' nuclear device, unless they pay a some of money.

Where it all started...

The OpenFPS project was begun by Chris Dalke on The Game Creators forum. Members of the online community signed up to form the current team.

The project is still in development, and updates are regularly posted in the forum thread.
