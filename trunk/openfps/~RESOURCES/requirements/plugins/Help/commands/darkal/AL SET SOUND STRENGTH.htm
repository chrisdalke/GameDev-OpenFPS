AL SET SOUND STRENGTH

This function will set how well the sound carries over distance. Note that this function does not work on 2D sounds, since no distance calculations take place when they are played.

  Syntax 
AL SET SOUND STRENGTH soundNumber, strength
  Parameters 
soundNumber
Integer
The ID you want to use to create this sound. 
strength
Float
The strength of the sound. Default value is 1.0. The higher the value, the better the sound will carry over distance and vice versa.

  Returns

This function does not return a value.
