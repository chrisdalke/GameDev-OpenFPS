AL SET SOUND INNER CONE ANGLE

This function will set the inner cone anlge of a sound. By default sounds are created with their inner and outer cone angles set to 360 degrees, which means them omnidirectional or 'point' sounds. A sound becomes a directional sound only if the inner cone angle is less than 360 degrees. If the listener is inside the inner cone, the sound is played at full volume. When the listener is inside the outer cone, the sound is attenuated based on the listener's distance from the sound and its distance from the edge of the inner cone. The diagram at the bottom of this page demonstrates how directional sounds work in darkAL.

  Syntax 
AL SET SOUND INNER CONE ANGLE soundNumber, angle
  Parameters 
soundNumber
Integer
The sound number. 
angle
Float
The angle of the inner cone. Ranges from 0.0 to 360.0 degrees.

  Returns

This function does not return a value.
